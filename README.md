## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Carlymaxx/maxxtechxmd.git
   cd maxxtechxmd
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set your SESSION_ID** and run:
   ```bash
   npm start
   ```
